Section I: Bezier Curves and Surfaces

Part 1: Bezier curves with 1D de Casteljau subdivision

De Casteljau algorithm is recursive way to linear interpolate control points with value t. When n points given, interpolate near two points p and p+1 with t:(1-t) ratio and make new n-1 points. Conduct this process till only one point left. The position of the last point is the point of bezier curve on parameter t.

[EvaluateStep 2D Function]
1. Check size of points vector and return points if size is 1.
2. If not, conduct loop for points.size()-1 times.
3. Conduct linear interpolating calculation [ p = (1 - t) * p_i + t * p_{i+1} ]
4. Return new points.

Part 2: Bezier surfaces with separable 1D de Casteljau

Bezier curve surface is defined with NxN control points and two parameter for row/column interpolate parameter. To find one point of bezier curve surface, conduct de Castelijau seperatly on row and column. Normally conduct on row first, and condcut column interpolation with result points of row interpolating.

[EvualteStep 3D Function]
Conduct interpolate one time on 3D space. Similar with part1, but difference is degree of vector. Return n-1 points.

[Evaluate1D Function]
Recursive bezier curve function. Conduct de Castelijau algorithm until only 1 point left with while loop. Return last 1 point of bezier curve.

[Evaluate Function]
The real curve surface evaluation function. Conduct 4 tiems row interpolation and 1 last column interpolation. Return 1 last point of bezier curve surfave.

Section II: Triangle Meshes and Half-Edge Data Structure

Part 3: Area-weighted vertex normals

The cross product give us the perpendicular vector of 2 given vectors and the size is twice of triangle with that two vectors. So if we know vectors of near triangles, we can conduct area-weighted interpolation with cross product. And we can get coordinates of near points easily with halfedge methods.

[Noraml Function]
1. Start with halfedge(), tour around the triangle with do-while and h->twin()->next().
2. Take coordinates of 3 points and make two vector.
3. Conduct cross product with that 2 vectors so that can get area-weighted vectors.
4. Accumulate every vectors on Vector3D variable n and return with normalized size.

Part 4: Edge flip

The edge flip make ugly divided quardrilateral to well divided quardrilateral with beutiful 2 triangle. setNeighbors() let us change the ugly edge to pretty edge easily. I drew the shape of triangles on my notepad and thought about how can I flip the edge.

[Edgeflip Function]
1. Save every entities into new variables such as halfedge, vertex, edge, face.
2. Use setNeighbors() so that change the elements of halfedges without defining new variable.
3. Reassign new halfedges of vertex, edge and faces.

Part 5: Edge split

The edge split let us split 2 triangles to 4 triangles. It's quite similar with edgeflip().

[Edgesplit Function]
1. Save every entities into new variables such as halfedge, vertex, edge, face.
2. Make new entities for new triangles. We need 3 more faces, 2 more edges, 1 more vertex and 6 more halfedges.
3. Use setNeighbors() so that change the elements of 12 halfedges.
4. Reassign new halfedges of 5 vertexes, 4 inside edges and 4 faces.

Part 6: Loop subdivision for mesh upsampling

Loop of edgeflip edgesplit to make mesh smoother.

[Upsample Function]
Step 1. For old vertives, calculate new position and save it on newPosition. Calculate value u with v, and interpolate it with position of neighbor positions.
Step 2. Calculate the split points of edges. for every edges, look over 2 triangels that contains edge and calculate the split point with interpolating 4 vertexes.
Step 3. Split the edges in real and give the new position from step 2 calculation. New vertexes are made isNew true.
Step 4. Flip the edges that connect old vertex and new vertex.
Step 5. Rotate all vertexes and set position to newPostions. Points move to new calculated position.
